# Prime_biblio 📚

Application full-stack de gestion et d'administration de bibliothèque moderne.

## 🚀 Technologies Utilisées

- **Backend :** Java 21, Spring Boot 3, Spring Security (JWT), JPA / Hibernate, PostgreSQL.
- **Frontend :** Angular 18+, Architecture basée sur les **Signals**, PrimeNG UI Component Library.

## 🛠️ Fonctionnalités

- 🔐 Authentification et autorisation sécurisées par rôles (Admin, Membre).
- 📊 Tableau de bord administrateur avec statistiques d'activité en temps réel.
- 🔑 Gestion dynamique des droits et création de rôles personnalisés.
- 📖 Catalogue de livres interactif avec flux d'emprunt et de réservation.
